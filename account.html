    1. How can I create a list?

    To create a list click on the button CREATE NEW MANUAL LIST, fill the form and then click SAVE.

    2. How can I add a user to the list?

    You can do this in three different ways:

    • On the Rank Screen;

    • When rate a task;

    • Through an automatic generated list click on the + icon.
    3. Can I create a list on the rate page?

    Yes. In the Rate page you just need to click on the "Add to list" link. If you have no lists yet the create list form will be shown.

    4. How can I create a job from list?

    Click on the POST JOB button. If you already have a list created, the option to create a job for list will be shown.
