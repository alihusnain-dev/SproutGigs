    Working with variables

    You can add variables to job steps if you want freelancers to see different descriptions. The variables in job steps will be replaced by values informed in the "Variables Values" field.

    A variable in job steps must be in the format {{VAR_NAME}} like the following example

    Then, inform the variables values separated by comma where the first line contains the names of the variables without curly braces like the image bellow

    Last, the proof requirement is not variable. So, make sure you ask for a proof that works for all informed values

    Every time a freelancer checks your job, the variables are replaced by the informed values randomly. Therefore, there is no guarantee that you will get proofs for all different values.
